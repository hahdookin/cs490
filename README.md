## :triangular_ruler: Architecture
```txt
.------------.               .------------.               .------------.
| Front-end  |   <---S--->   | Middleware |   <---S--->   |  Back-end  |
'------------'               '------------'               '------------'
    /|   POST (Fetch)                                           /|
     |                                                           |  SQL
     |/  JSON                                                    |/
.------------.                                            .------------.
|  Browser   |                                            |  Database  | 
'------------'                                            '------------'

========================================================================
Legend:
     
<---S--->
^1      ^2

^1 - JSON
^2 - POST (Curl)
```
